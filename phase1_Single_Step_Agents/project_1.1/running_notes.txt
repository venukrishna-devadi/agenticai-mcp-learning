CORE IDEA OF THIS PROJECT

The agent must decide:
Should I:
- summarize?
OR
- generate detailed breakdown?
This may look simple.

But conceptually, this is massive.

Because now the LLM is no longer:

just generating text

It is:

selecting actions.

That is the foundation of:

* tool use
* MCP
* LangGraph routing
* multi-agent systems

⸻

WHAT YOU ARE LEARNING HERE

This project introduces:

Concept 1 — Intent Routing

The model determines:

what action should happen.

⸻

Concept 2 — Tool Invocation

The model does not directly answer.

Instead:

* selects a tool
* tool performs work
* agent returns result

This separation is critical.

⸻
