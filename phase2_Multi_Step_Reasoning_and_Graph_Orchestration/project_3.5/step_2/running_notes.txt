📊 CONFIDENCE-BASED ROUTING
┌─────────────────────────────────────────────────────────────────────────────────────┐
│                    CONFIDENCE + CONTEXT DECISION MATRIX                              │
└─────────────────────────────────────────────────────────────────────────────────────┘

Confidence →  0.0-0.3        0.3-0.6        0.6-0.8        0.8-1.0
─────────────────────────────────────────────────────────────────────────────────────

Tool Available = YES + Ambiguity = LOW
─────────────────────────────────────────────────────────────────────────────────────
               ASK HUMAN      TRY TOOL       TOOL OK        DIRECT
                             (with note)    (preferred)

Tool Available = YES + Ambiguity = HIGH
─────────────────────────────────────────────────────────────────────────────────────
               ASK HUMAN      ASK HUMAN      TOOL +         DIRECT
                                            VERIFY         (with source)

Tool Available = NO + Ambiguity = LOW
─────────────────────────────────────────────────────────────────────────────────────
               ASK HUMAN      DIRECT         DIRECT         DIRECT
                             (low conf)     (ok)

Tool Available = NO + Ambiguity = HIGH
─────────────────────────────────────────────────────────────────────────────────────
               ASK HUMAN      ASK HUMAN      ASK HUMAN      DIRECT
                                                                 (rare)

LEGEND:
├── ASK HUMAN → need_human = True
├── TRY TOOL → need_tool = True (with fallback)
├── TOOL OK → need_tool = True
├── DIRECT → need_tool = False, need_human = False
└── VERIFY → Use tool, then double-check with human
