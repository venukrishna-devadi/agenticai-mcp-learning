PROJECT 3.5 — PHASE 1
Human-in-the-Loop (HITL) ReAct Agent

GOAL OF PHASE 1
We want the agent to:
1. Think
2. Decide if uncertain
3. Ask human clarification
4. Receive human input
5. Continue reasoning
6. Produce final answer

WHAT WE ARE NOT ADDING YET
NOT YET:

* audit trails
* approvals
* policy engines
* overrides
* compliance

ONLY HITL interruption + resume.

CORE CONCEPT OF THIS PROJECT
This project introduces: INTERRUPTIBLE AGENTS

Meaning:
the graph can pause and ask for help.
